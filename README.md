# Bolado Browser

This is a project of an Open Source Microsoft Windows browser made with C#, .NET and WPF for learning purposes

## Technologies

- [Microsoft Windows](https://www.microsoft.com/en-us/windows)
- [C#](https://dotnet.microsoft.com/en-us/languages/csharp)
- [.NET 8](https://learn.microsoft.com/en-us/dotnet/core/whats-new/dotnet-8/overview)
- [WPF](https://learn.microsoft.com/en-us/dotnet/desktop/wpf/overview/?view=netdesktop-8.0)
- [WebView2](https://learn.microsoft.com/en-us/microsoft-edge/webview2/landing/)
- [Git](https://git-scm.com/)
- [Visual Studio](https://visualstudio.microsoft.com/)
- [MVVM](https://learn.microsoft.com/en-us/archive/msdn-magazine/2009/february/patterns-wpf-apps-with-the-model-view-viewmodel-design-pattern)
- [MVVM Toolkit](https://learn.microsoft.com/en-us/dotnet/communitytoolkit/mvvm/)
- [Rx.NET](https://github.com/dotnet/reactive)
